1. About POCO Launcher

POCO Launcher is a full-featured smartphone system made by Xiaomi. Being lightweight, it can boast high performance and multiple personalization settings. At the same time, POCO Launcher provides a complete set of security solutions to ensure a more private usage environment. Here are the main features of POCO Launcher:

1.1 Applications/Apps Management
POCO Launcher provides users with a variety of ways to organize and search for apps on their devices. For example, all apps are stored in the App drawer, so the Home screen looks neat and tidy. Users also receive smart recommendations to install the apps they might need. Recently installed and used apps are displayed in the top part of the drawer which makes finding them a lot easier. Apps are automatically grouped according to their categories as well. Apps can also be grouped by the color of their icon, which is an unusual but very effective way to find apps.

1.2 Desktop personalization
POCO Launcher offers lots of options when it comes to personalizing the desktop. During the initial setup, users can select beautiful wallpapers. They can also browse, download, and use different themes and wallpapers in our theme store. Themes also allow to replace existing icon packs with something much more exciting. Arranging apps on desktop is gesture-friendly: you can filter and drag apps to rearrange them in any way you want.

1.3 Basic system features
Basic system features become more convenient and entertaining with POCO Launcher: crisp Clock widget shows time and date so you can always see them, necessary toggles are always one tap away, and notifications allow you to see the unread messages right away.

1.4 Privacy and security
Among all the privacy and security-related features of POCO Launcher, there is one that allows you to hide specific apps so that only you could see and use them.
